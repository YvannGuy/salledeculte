# Plan netlinking 30/60/90 jours – salledeculte.com

## Objectifs

- Augmenter l’autorité du domaine de façon **white-hat**.
- Prioriser les pages : **Home** `/`, **/rechercher**, pages locales (ville/département), guides piliers.
- Mix d’ancres : **brand/URL** majoritaires ; quelques ancres thématiques souples ; très peu d’ancres “exact match” money.

---

## 1. Données de départ (à consolider)

1. **Inventaire actuel** (Ahrefs / Semrush / GSC) :
   - DR/AS du domaine.
   - Nombre de domaines référents → objectif : **+20 domaines de qualité** sur 90 j.
2. **Analyse d’ancres** :
   - Répartition brand / URL / money.
   - Nettoyer PBN / spam évidents ; désaveu GSC si nécessaire.
3. **Link Intersect** avec 3–5 concurrents → liste de domaines à cibler.

---

## 2. Types de spots visés

- **Associations / fédérations religieuses** : pages ressources, partenaires, annuaires de lieux de culte.
- **Blogs & médias chrétiens / religieux / associatifs** : articles invités, interviews, études de cas.
- **Sites événementiel / mariage / séminaires** : pages “ressources” ou “outils pour organiser un événement”.
- **Annuaires qualitatifs** : plateformes B2B, SaaS pour associations (pas fermes de liens).
- **Presse locale / régionale** : communiqué sur le rôle de la plateforme.
- **Partenariats fournisseurs** : sonorisation, traiteur, sécurité → liens croisés “nos partenaires”.

---

## 3. Calendrier 30/60/90 jours

### J+0 à J+30 : fondations & quick wins

**Objectifs** : Nettoyage du profil, premiers liens faciles, ressources linkables.

| Semaine | Action |
|--------|--------|
| 1 | Export backlinks → désaveu domaines TLD exotiques, annuaires spammy, PBN. Mise à jour disavow GSC. |
| 2–4 | Contacter associations utilisatrices pour **lien page partenaires**. Partenaires actuels pour ajout lien. Inscription **3–5 annuaires qualitatifs FR** (associatif / SaaS / pro événementiel). |

**Volumes** : 8–10 nouveaux domaines référents.

**Ancres** : 70 % brand/URL, 20 % neutres (“voir le site”, “plateforme”), 10 % thématiques douces (“plateforme de location de salles de culte”).

**Pages cibles** : Home (majorité), 1–2 liens vers /rechercher, 1 lien vers un guide pilier dès publication.

---

### J+30 à J+60 : montée en puissance éditoriale

**Objectifs** : Liens éditoriaux contextuels.

| Semaine | Action |
|--------|--------|
| 5–8 | **Guest posts** : identifier 20 sites FR (blogs religieux, associatifs, événementiel) via Link Intersect + Google. Proposer articles invités (“Comment les communautés trouvent des lieux pour leurs cultes en ville ?”), études de cas. |
| 5–8 | **Relations presse** : pitch 3–5 médias locaux/régionaux (Paris, IDF) – “Une plateforme qui aide les communautés à trouver des lieux de culte temporaires”. |

**Volumes** : 10–15 nouveaux domaines référents.

**Ancres** : 60–65 % brand/URL, 25–30 % partiellement optimisées (“plateforme de salles pour cultes”), ≤10 % exact match (à utiliser très rarement).

**Pages cibles** : 50 % guides piliers & pages locales (Paris, 93), 30 % home, 20 % /avantages.

---

### J+60 à J+90 : consolidation & assets linkables

**Objectifs** : Créer des contenus qui méritent des liens ; poursuivre l’éditorial.

| Action |
|--------|
| **Assets** : Étude “État des lieux des besoins en salles de culte en Île-de-France” (données anonymisées) ; checklist PDF pour associations → page dédiée. |
| **Sensibilisation** : Contacter blogueurs / médias pour soumettre l’étude. |
| **Partenariats** : Programmes d’affiliation ou co-marketing avec fédérations, réseaux d’églises, associations. |

**Volumes** : 10–15 nouveaux domaines référents.

**Ancres** : Profil très brandé (≥70 %).

**Pages cibles** : 40 % page étude/ressource, 30 % home, 30 % /rechercher + pages locales.

---

## 4. Données à extraire (Ahrefs / Semrush / GSC)

- **Ahrefs/Semrush – Site Explorer** :
  - Referring domains (filtrer DR/AS ≥ 20).
  - Backlinks : dofollow, FR, pages avec trafic.
  - Rapport **Anchors** : catégoriser Brand / URL / Money.
- **GSC** : Liens externes, “Pages les plus liées”.
- **Filtres** : Exclure spam score élevé, TLD exotiques ; privilégier liens contextuels.

---

## 5. Détection sur-optimisation d’ancres

- **Cibles** : 50–70 % brand, 10–20 % URL, ≤10–15 % money.
- **Risques** : Beaucoup de liens faible DR avec ancres money exact match ; pics de liens depuis sites douteux.
- **Action** : Rééquilibrer par des liens brand/URL sur les prochaines campagnes.

---

## 6. 20 opportunités prioritaires (méthode)

1. **Link Intersect** (Ahrefs/Semrush) : 3–5 concurrents → domaines qui linkent ≥2 concurrents mais 0 vers toi.
2. Classer par DR/AS et trafic organique.
3. Prioriser : FR/EU, thématiques proches (événementiel, associatif, religieux).
4. Compléter avec sites ayant déjà cité “location salle culte” / “événement religieux” dans leurs contenus.

---

## 7. Monitoring

- **Mensuel** : DR/AS, nombre de domaines référents, trafic orga, positions sur 10 mots-clés cibles. Vérifier qu’aucun cluster d’ancres money ne dépasse ~10–15 %.
- **Trimestriel** : Réévaluer pages cibles ; nouveaux concurrents ; relancer Link Intersect.
